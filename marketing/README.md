Everything in this folder is generated by the scripts in [`scripts/`](./scripts).
You can regenerate the videos and the pitch deck on **macOS, Linux, or Windows** —
the pipeline has **no Mac-only dependencies**.

## Quickstart (any OS)

Prereqs:

- **Python 3.9+** (`python3 --version`)
- **ffmpeg** in `PATH` ([install guide](https://ffmpeg.org/download.html) — `brew install ffmpeg` on macOS, `apt install ffmpeg` on Debian/Ubuntu, `choco install ffmpeg` on Windows)
- **Internet access** the first time you run anything (so `edge-tts` can reach Microsoft's free neural TTS endpoint, and `fetch_fonts.py` can pull the OFL fonts)

```sh
# 1. Install Python deps (cross-platform)
pip install -r marketing/scripts/requirements.txt

# 2. Download the bundled fonts (Inter + JetBrains Mono, both OFL 1.1)
python3 marketing/scripts/fetch_fonts.py

# 3. Build the artefacts
python3 marketing/scripts/build_demo.py            # → marketing/demo.mp4
python3 marketing/scripts/build_pitch_video.py     # → marketing/investor-pitch.mp4
python3 marketing/scripts/build_deck.py            # → marketing/pitch-deck.pdf
```

That's it. Each build script is idempotent — re-run as many times as you want.

## Voiceover backends

`scripts/tts.py` picks the first available backend in this order:

| Priority | Backend | Setup | Quality | Network? |
| --- | --- | --- | --- | --- |
| 1 | [`edge-tts`](https://github.com/rany2/edge-tts) | `pip install edge-tts` | Neural (best) | Yes |
| 2 | [`pyttsx3`](https://pypi.org/project/pyttsx3/) | `pip install pyttsx3` | Native OS (decent) | No |
| 3 | macOS `say` | built-in (Darwin only) | Native (decent) | No |

Force a specific backend at any time:

```sh
PRODUCTIVEYOU_TTS_BACKEND=pyttsx3 python3 marketing/scripts/build_demo.py
PRODUCTIVEYOU_TTS_BACKEND=mac_say python3 marketing/scripts/build_demo.py
```

To swap in a **paid commercial voice** (ElevenLabs, OpenAI TTS, Google Cloud), edit `synthesize()` in [`scripts/tts.py`](./scripts/tts.py) — the build scripts call it with a stable `(text, out_path, voice, rate)` signature.

## Fonts

Bundled (not depended on system fonts):

| Font | Use | License |
| --- | --- | --- |
| **Inter** (variable, 100–900 weights) | Slide titles, body, captions | [SIL OFL 1.1](./scripts/fonts/OFL.txt) — © Rasmus Andersson |
| **JetBrains Mono** (Regular + Bold) | Kickers, URLs, mono labels | SIL OFL 1.1 — © JetBrains |

The fonts are **committed alongside the scripts** (~2.3 MB total) so builds are fully offline-reproducible. If you ever delete the `scripts/fonts/` folder, `fetch_fonts.py` will re-download them on next run using only Python stdlib.

## Design system

`scripts/slidekit.py` mirrors the app's Tailwind tokens:

| Token | Value |
| --- | --- |
| Background | `hsl(240 10% 4%)` |
| Foreground | `hsl(45 15% 90%)` |
| Accent | `hsl(38 95% 52%)` → `hsl(32 80% 42%)` |
| Muted | `hsl(220 8% 42%)` |
| Type | Inter (variable, 100–900), JetBrains Mono |
| Canvas | 1920 × 1080 (16:9) |

```
    ├── build_pitch_video.py    # investor narrative video

```
