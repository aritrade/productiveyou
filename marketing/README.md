# Marketing Assets

Everything in this folder is generated by the scripts in [`scripts/`](./scripts).
You can regenerate the videos and the pitch deck at any time — no proprietary
tools required. macOS only (uses the built-in `say` voice and Helvetica Neue).

## Contents

| File | Purpose | Length / Size |
| --- | --- | --- |
| [`demo.mp4`](./demo.mp4) | 90-second product demo for the README and landing page | ~88s · 2.3 MB |
| [`investor-pitch.mp4`](./investor-pitch.mp4) | 2.5-minute narrated investor walkthrough | ~143s · 3.7 MB |
| [`pitch-deck.pdf`](./pitch-deck.pdf) | 15-slide investor deck (problem → solution → TAM → economics → ask) | 15 pages · 1.7 MB |
| [`thumbnails/`](./thumbnails) | Static PNG cover frames + voiceover manifests | — |

## Regenerate everything

Prereqs: macOS, Python 3.9+, [`ffmpeg`](https://ffmpeg.org), and
`pip install pillow reportlab` (reportlab is reserved for future expansion;
the deck PDF is currently built via Pillow's native PDF writer).

```sh
cd marketing/scripts

# 90s product demo
python3 build_demo.py            # → marketing/demo.mp4

# 2.5min investor narrative
python3 build_pitch_video.py     # → marketing/investor-pitch.mp4

# 15-slide pitch deck
python3 build_deck.py            # → marketing/pitch-deck.pdf
```

## Voiceover

Uses macOS `say` with the **Samantha** voice at 170–175 wpm. Switch to a
neural voice by changing the `VOICE` constant in either build script, or
swap in an external TTS API (ElevenLabs, OpenAI, Google) by replacing
`synthesize_audio()`.

## Design system

The slide kit (`scripts/slidekit.py`) mirrors the app's Tailwind tokens:

| Token | Value |
| --- | --- |
| Background | `hsl(240 10% 4%)` |
| Foreground | `hsl(45 15% 90%)` |
| Accent | `hsl(38 95% 52%)` → `hsl(32 80% 42%)` |
| Muted | `hsl(220 8% 42%)` |
| Type | Helvetica Neue (titles), Menlo (mono) |
| Canvas | 1920 × 1080 (16:9) |
